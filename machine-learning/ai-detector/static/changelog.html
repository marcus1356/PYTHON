Lançamento inicial da plataforma. Estrutura completa de backend para detecção de conteúdo gerado por IA, com motor heurístico de análise, API REST versionada, persistência assíncrona e suite de testes abrangente.
✦ Funcionalidades
  • API REST completa para submissão e análise de textos API
  • Motor heurístico heuristic-v1 com 5 métricas: comprimento médio de frase, riqueza vocabular, burstiness, densidade de pontuação e comprimento médio de palavra
  • Veredicto em 3 níveis: human (<0.40), uncertain (0.40–0.60), ai (>0.60)
  • Confiança da predição em 3 níveis: low, medium, high
  • Re-análise forçada via parâmetro force_reanalyze=true
  • Paginação e filtros por status e fonte na listagem de submissions
  • Sandbox interativo para testes manuais via browser
⬡ Infraestrutura
  • FastAPI 0.115 com lifespan moderno (substituiu @on_event)
  • SQLAlchemy 2.0 totalmente assíncrono via aiosqlite DB
  • Alembic configurado para migrações de schema
  • Arquitetura em camadas: Endpoints → Services → Repositories → Database
  • Injeção de dependências via Annotated[AsyncSession, Depends(get_db)]
  • CORS habilitado para todos os origens (ambiente de desenvolvimento)
◎ Testes
  • 65 testes automatizados cobrindo endpoints, serviços, schemas e heurísticas
  • Banco SQLite em memória para isolamento total entre testes
  • Dependency override do FastAPI para injetar sessão de teste
  • Cobertura de 87% mensurada com pytest-cov
  • Relatório HTML de cobertura gerado em htmlcov/
  • 4 suítes independentes: endpoints, serviço, schemas, heurísticas
